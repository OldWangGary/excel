设定简单，可单独生成单一运算题，也可生成加减、乘除混合题，合适学前班以及一、二年级学生使用

直接打印即可，默认设定为A4纸，解除工作表保护后，可自行排版，也可选打印答案，供宝宝自行核对
设定自带提示，上手简单
